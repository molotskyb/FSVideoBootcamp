# Course Map

This repository supports the Full-Stack Video Streaming Bootcamp.

The project is designed as a single evolving codebase. Each course part adds a new layer to the same practical streaming platform.

---

## Part 1. Encoding & Packaging Foundations

Status: published

### Module 1. FFmpeg Encoding Foundations

Focus:

- ffprobe inspection
- MP4 transcoding
- GOP/keyframe settings
- CRF, CBR, and 2-pass VBR
- ABR ladder creation

### Module 2. Packaging Foundations

Focus:

- HLS CMAF
- DASH CMAF
- Shaka Packager
- captions and thumbnails
- manifest validation

---

## Part 2. Content Delivery, CDN, QoE, Canary, and DRM

Status: in progress

### Module 1. Content Delivery. S3 + CDN Fundamentals

Focus:

- S3 static website hosting
- CloudFront distribution setup
- signed URLs
- signed cookies
- cache key behavior
- invalidation
- versioning
- OriginPath rollback
- QoE logging
- 480p canary stream
- mini-review and git checkpoint

### Module 2. DRM

Focus:

- DRM landscape
- Widevine / PlayReady / FairPlay overview
- ClearKey
- CENC
- KID / KEY
- PSSH
- license request flow
- Widevine development playback
- dash.js protection data
- DRM error interpretation
